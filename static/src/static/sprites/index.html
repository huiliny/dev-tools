使用说明：

1、将所有需要合并的图片放在目录./src/static/sprites；

2、运行命令guilin sprites即可将所有指定的图片合并为sprites.png；新增图片时执行命令即可；

3、仅支持jpg和png两种格式，并且不支持中文命名；

4、引入生成样式或将红色样式复制到项目样式，在需显示图标位置添加对应class
